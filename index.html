Project Description

INDIMEET is a full-stack Tinder-style matchmaking app built with the MERN stack. It provides a fast, mobile-friendly swipe experience where users can discover profiles, match, and start conversations in real time. The app focuses on smooth UX, secure authentication, and reliable realtime messaging.

Key features include swipe & matching mechanics, profile cards and image uploads, JWT-based authentication (access & refresh tokens), Axios for API interaction, and WebSocket-powered chat for instant messaging. The backend enforces authorization for protected routes and stores user/state data in MongoDB.

Technologies: MongoDB, Express, React, Node (MERN), Axios, JSON Web Tokens (JWT), WebSockets (Socket.io or ws), Deployment (AWS).
